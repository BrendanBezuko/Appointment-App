# Clinician Appointment Application

## Running

### Database

1. insure sqlite3 is installed on your system.
1. Create database `cd backend && qlite3 ./db/main.db < ./db/database.sql`

### Backend

1. Install dependencies `npm install`
2. Run backend `npm start`
